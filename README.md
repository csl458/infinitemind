# Mind Brain

一个本地优先的无限画布知识脑图应用。你可以把 PDF 放进画布，在节点里写摘要、问题、推导和行动项，逐步构建自己的知识网络。

## 已实现能力

- 无限画布：平移、缩放、拖拽节点、手动连线。
- 文本笔记卡：适合记录总结、反驳点、实验计划和灵感。
- PDF 卡片：支持按钮导入，也支持直接把 PDF 拖进画布。
- 本地持久化：节点和连线保存在浏览器存储中，PDF 文件保存在 IndexedDB。
- 导出和导入：可以把整张脑图导出成 JSON，再完整恢复。

## 技术栈

- React 19
- Vite 8
- @xyflow/react
- react-pdf + pdfjs-dist
- localforage

## 本地运行

```bash
npm install
npm run dev
```

默认开发地址由 Vite 提供，通常是 http://localhost:5173。

## 从桌面进入

- 我已经在桌面放了一个可双击启动的入口：Mind Brain.cmd。
- 双击后会自动检查本地服务是否已经启动；如果没有，它会拉起项目并打开浏览器。
- 工作区里的实际启动脚本在 [launch-mind.ps1](launch-mind.ps1)。如果桌面入口丢了，重新用它生成一个快捷方式即可。

## 生产构建

```bash
npm run build
```

构建结果会输出到 dist 目录。

## 使用说明

1. 点击“新建笔记卡”可以插入一个空白思考节点。
2. 点击“导入 PDF”或直接拖入 PDF 文件，会生成带预览的 PDF 节点。
3. 通过节点上下方的连接点拖拽连线，组织你的知识结构。
4. 点击“导出脑图”会生成一个包含图结构和 PDF 数据的 JSON 文件。
5. 点击“导入脑图”可以恢复之前导出的完整工作区。

## 当前限制

- PDF 节点默认展示第一页预览，适合做文献入口和摘要锚点。
- 导出文件会内嵌 PDF 数据，文件体积会随着文档数量增长。
- pdf.js 会带来较大的前端包体积，但本地运行和构建已经正常通过。
